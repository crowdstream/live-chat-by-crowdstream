== Installation ==

1. Download Live Chat by Crowdstream plugin
2. Unarchive Live Chat Support plugin
3. Copy the folder with Live Chat by Crowdstream plugin
4. Open ftp \wp-content\plugins\ Live Chat by Crowdstream
5. Paste the plug-ins folder in the folder
6. Go to admin panel => open item "Plugins" => activate Live Chat by Crowdstream plugin
7. Create a Crowdstream account http://www.crowdstream.io/signup
8. Follow the instructions and copy your APP ID
9. Paste your APP ID into admin panel => "Plugins" => "Live Chat by Crowdstream" => "Your Crowdstream APP ID"
10. Click Save.
